# 🎫 Customer Support Ticket System

An Customer Support Ticket System that automates the classification and prioritization of customer queries using Machine Learning and Natural Language Processing (NLP). The system analyzes support tickets, predicts their category, assigns a priority level, and generates an appropriate response through a user-friendly web interface.

---

## 🚀 Features

- Automated ticket classification using Machine Learning
- Priority prediction for incoming support requests
- NLP-based text preprocessing and analysis
- Real-time ticket processing
- Automated email notifications
- Responsive web interface
- Flask-powered backend integration

## 📂 Project Workflow

1. User submits a support ticket.
2. The ticket text is preprocessed using NLP techniques.
3. The trained ML model predicts the ticket category.
4. The system assigns a priority level.
5. A suitable response is generated and displayed.
6. An email notification is sent to the user.
